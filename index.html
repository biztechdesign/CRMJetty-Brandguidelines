02

Logo Misuse

Never alter, distort, or misrepresent the CRMJetty logo. Consistent, correct usage is essential to brand trust.

Do Not Stretch

Never stretch or compress the logo horizontally or vertically

Do Not Rotate

Never rotate or tilt the logo at any angle

Do Not Recolour

Never use unapproved colours or partial colour changes

No Heavy Shadows

Never apply drop shadows, glows, or heavy effects

Low Contrast BG

Never place the coloured logo on clashing or low-contrast backgrounds

No Outlines

Never add strokes, outlines, or borders to logo elements

No Rearranging

Never rearrange, stack, or separate logo elements arbitrarily

No Old Logo

Never use outdated, legacy, or unauthorised logo versions

Clashing Colour

Never place the logo on red, orange, or other high-contrast clashing backgrounds

Grayscale Version

For greyscale or B&W documents — single-colour printing, embossing, and fax

Style Rules

Stroke-based, outlined. Stroke width: 1.8px. Caps and joins: round. No fill. Corner radius: 2px on rectangles.

Sizes & Grid

Design on 24×24px grid. Use at 16px, 20px, 24px, 32px, 48px. Never scale below 14px — switch to filled icon style at small sizes.

Color Rules

Default: #324FC3 (primary blue). On dark bg: rgba(255,255,255,0.85). In buttons: inherit button text color. Disabled: #9CA3AF.

09

Do's & Don'ts

Practical rules that ensure every team member, agency, and partner uses the CRMJetty brand consistently and correctly.

Logo — Do

  • Always use the approved logo file from the official brand kit
  • Maintain clear space equal to the height of the "J" on all sides
  • Use the white logo on dark and coloured backgrounds
  • Use the full-colour logo on white and light backgrounds
  • Scale the logo proportionally from a corner
  • Use SVG for digital and EPS/PDF for print

Logo — Don't

  • Never stretch, skew, or distort the logo in any direction
  • Never rotate or tilt the logo
  • Never recolour the logo with unapproved colours
  • Never apply drop shadows, glows, or bevel effects
  • Never place on a low-contrast or clashing background
  • Never rearrange or separate the icon from the wordmark

Color & Typography — Do

  • Use #022484 as the dominant brand colour (60% rule)
  • Maintain WCAG AA contrast ratio (≥4.5:1) for all text
  • Use Inter across all touchpoints — UI, body, forms, and display headings
  • Follow the type scale — do not freestyle font sizes
  • Use semantic colours (success/error) only for their function
  • Use approved gradients for backgrounds and CTAs

Color & Typography — Don't

  • Do not use green, orange, or yellow as brand accent colours
  • Never use pure black (#000000) for primary text
  • Do not mix more than two font families in one layout
  • Never use all-caps for long body copy passages
  • Do not reduce font size below 12px for body content
  • Never use colour alone to communicate status — always add a label

UI & Website — Do

  • Use btn-primary for the single main CTA on any page
  • Ensure all interactive elements have a visible focus ring
  • Use card hover states to indicate clickable elements
  • Apply consistent 8px grid spacing throughout layouts
  • Test on mobile — all components must be touch-friendly (44px min tap)

UI & Website — Don't

  • Never use more than two primary CTAs side by side on a page
  • Do not remove focus rings for accessibility compliance
  • Never use inline styling that overrides design token values
  • Do not mix card styles inconsistently in the same grid
  • Never use placeholder text as a substitute for form labels
